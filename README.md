# Scorekeepapp

#Features:
- Choose 9 or 18 holes, and from 1-4 player with editable a,esat the start
- Add the per hole strokes use a keyboard or numeric keypad.
- Auto-calculated totals per player
- Allows for undoing lat entry, clearing current hole, and a full reset which takes you to the starting page
- No backend needed

- Index.html
- styles.css
- app.js
- ai-notes.md
